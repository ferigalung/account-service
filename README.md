# Account Service

## Deskripsi

Sebuah service dimana nasabah dapat membuat akun dan melakukan transaksi.
Ini adalah project assessment untuk Ihsan Solusi.

## Fitur

- Create Account
- Get Account Balance
- Withdraw Balance
- Deposit Balance
- Get Transaction History

## Tech Stack

- Go
- Fiber
- PGX
- PostgreSQL
- Docker
